# Airbnb_ML_Project

Final Project for Break Through Tech AI's Machine Learning Foundations Course

This project leverages Airbnb New York City listings data to predict the number of guests a listing can accommodate. As a supervised learning and regression task, the project employs various models, including Linear Regression, Decision Tree Regressor, Stacking Regressor, Gradient Boosted Trees, and Random Forest.
